# Calculator App

A fully functional calculator built using JSP, Servlets, and CSS. It supports basic arithmetic operations (+, -, *, /), includes a clear button, and handles errors like division by zero. The UI is responsive and interactive, with bonus features like calculation history and keyboard support.

## Features
- Basic arithmetic operations
- Clear button
- Error handling for invalid inputs
- Calculation history
- Keyboard support

## Code Explanation
- **JSP**: Handles the UI and form submission.
- **Servlet**: Processes the calculations and manages history.
- **CSS**: Provides a clean and responsive design.

## Challenges Faced
- Implementing keyboard support required JavaScript event listeners.
- Managing calculation history in a session was initially tricky.

## Learnings
- Improved understanding of JSP-Servlet interaction.
- Gained experience in creating responsive UIs with CSS.
